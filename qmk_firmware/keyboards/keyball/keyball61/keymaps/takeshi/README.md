# takeshi キーマップの設定情報

このディレクトリは、Keyball61のカスタムキーマップ `takeshi` の設定ファイルを格納しています。

## 同時押し（コンボキー）設定一覧

本キーマップでは、特定のキーを同時に押すことで、別のキー入力をシミュレートする「コンボキー（Combos）」機能が有効化されています。設定されている組み合わせは以下の通りです。

| コンボ名 | 同時押しキー組み合わせ | 出力されるキー | 機能・用途 |
| :--- | :--- | :--- | :--- |
| `my_F2` | `Space (Layer 2)` + `W` | `F2` | F2キーの入力（ショートカットなど） |
| `combo_df` | `D` + `F` | `ESC` | 左手のホームポジションのままでEscを入力 |
| `combo_jk` | `J` + `K (Layer 3)` | `Enter` | 右手のホームポジションのままでEnterを入力 |
| `combo_zxc` | `Z` + `X` + `C` | `MUTE` | メディアの音量を消音（ミュート） |
| `combo_sd` | `S` + `D` | `Tab` | 左手のホームポジションのままでTabを入力 |
| `combo_kl` | `K (Layer 3)` + `L` | `Backspace` | 右手のホームポジションのままでバックスペースを入力 |

---

## その他のカスタマイズ特徴
- **Caps Word機能の有効化**: 左右のシフトキーを同時に押すなどして有効化される、大文字入力自動化機能（Caps Word）に対応しています。
- **Remap/VIA対応**: Webブラウザ経由で直感的にキーマップをカスタマイズ可能なRemapやVIAに対応しています。
- **トラックボールのスクロールモード**: レイヤー3がアクティブな間、トラックボールを動かすとマウススクロールになるように設定されています。
- **レイヤー連動LED制御**: 親指/トラックボール部のLEDのみ、アクティブなレイヤーに応じて色が変化します（マスター主導での同期制御）。
